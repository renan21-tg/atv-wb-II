# Requisitos

Node.js: v22.14.0

# Instalação

Clone o repositório:

    git clone https://github.com/renan21-tg/atv-wb-ii.git
    cd atv-wb-ii

Instale as dependências:

    npm install

# Executando o Projeto

    npm run dev

Abra http://localhost:3000 em seu navegador para visualizar a aplicação.
